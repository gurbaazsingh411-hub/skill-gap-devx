# Skill Gap by DevX

Master the skills for your dream role. Skill Gap uses AI to analyze your resume against target roles, identifies missing skills, and provides a personalized learning roadmap.

## Features

- **AI Resume Analysis**: Upload your resume and get instant feedback on how you match up against specific industry roles.
- **Skill Gap Identification**: See exactly which technical and soft skills you are missing.
- **Industry Benchmarking**: Compare your skill profile against industry averages.
- **Personalized Roadmaps**: Get a structured learning plan with curated resources (courses, documentation, videos) to bridge your gaps.

## Technologies

- **Frontend**: React, TypeScript, Vite, Tailwind CSS, shadcn/ui, Framer Motion
- **Backend / AI**: Supabase Functions, Google Gemini 2.5 Flash
- **Icons**: Lucide React

## Getting Started

### Prerequisites

- Node.js & npm (or bun)
- Supabase account (for AI functions)

### Running Locally

1. Clone the repository:
   ```sh
   git clone https://github.com/gurbaazsingh411-hub/skill-gap-devx.git
   cd skill-gap-devx
   ```

2. Install dependencies:
   ```sh
   npm install
   ```

3. Set up environment variables:
   Create a `.env` file with your Supabase credentials.

4. Start the development server:
   ```sh
   npm run dev
   ```
